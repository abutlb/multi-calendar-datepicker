Multi Calendar Datepicker

Vanilla JS · No dependencies · Gregorian, Hijri & Jalali · RTL & Arabic · CDN-ready

30-Second Quickstart

  1. 1 Include the CSS and JS
    <link rel="stylesheet" href="multi-calendar-datepicker.css">
    <script src="multi-calendar-datepicker.js"></script>
  2. 2 Add an input element
    <input type="text" id="mydate" readonly>
    <input type="hidden" id="mydate_val" name="date">
  3. 3 Initialize with one call
    MultiCalendarDatepicker.init('#mydate', { calendar: 'hijri', locale: 'ar', ... });
  4. 4 Done — the hidden input always receives Gregorian ISO
    No manual date conversion needed. name="date" submits 2026-06-15 to your server regardless of the displayed calendar.

Minimal Code

<!-- 1. Load files -->
<link rel="stylesheet" href="multi-calendar-datepicker.css">
<script src="multi-calendar-datepicker.js"></script>

<!-- 2. HTML -->
<input type="text"   id="mydate"     readonly>
<input type="hidden" id="mydate_val" name="date">

<!-- 3. Init -->
<script>
  MultiCalendarDatepicker.init('#mydate', {
    calendar:     'gregorian',  // 'gregorian' | 'hijri' | 'jalali'
    locale:       'en',         // 'en' | 'ar'
    outputTarget: '#mydate_val',// hidden input gets Gregorian ISO
    resultTarget: '#myres',     // optional: show result text here
    resultTemplate: 'Selected: {{gregorian}}',
  });
</script>
