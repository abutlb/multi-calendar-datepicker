Code

MultiCalendarDatepicker.init('#gdate', {
  calendar:        'gregorian', // 'gregorian' | 'hijri' | 'jalali'
  locale:          'en',        // 'en' | 'ar'
  dir:             'ltr',       // 'ltr' | 'rtl'
  weekStart:       0,           // 0=Sun, 1=Mon, 6=Sat
  outputTarget:    '#gdate_val',// hidden input → DB value
  resultTarget:    '#gres',     // element to show result text
  resultTemplate: 'Selected: {{gregorian}}',
});

Available Template Variables — المتغيرات المتاحة

  • {{gregorian}} ISO date sent to the database — e.g. 2026-06-15
  • {{display}} Human-readable date in the chosen calendar system
  • {{year}} Year component of the selected date (in display calendar)
  • {{month}} Month number
  • {{day}} Day number
  • {{monthName}} Full month name in the active locale (e.g. "June" / "يونيو")
