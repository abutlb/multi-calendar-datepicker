كيف يعمل: يعرض التقويم التواريخ بالتقويم الجلالي/الشمسي الفارسي. عند الاختيار، يُحفظ في #jdate_val التاريخ الميلادي بصيغة ISO (YYYY-MM-DD) تلقائياً — لا تحتاج إلى أي تحويل يدوي.

How it works: Calendar displays Jalali (Shamsi) dates. On selection, the hidden input automatically receives the Gregorian ISO date — zero manual conversion needed.

Code

MultiCalendarDatepicker.init('#jdate', {
  calendar:       'jalali',  // Jalali / Shamsi / Persian
  locale:         'ar',      // 'ar' for Arabic names | 'en' for English names
  dir:            'rtl',
  outputTarget:   '#jdate_val',
  resultTarget:   '#jres',
  resultTemplate: 'الجلالي: {{display}} | الميلادي: {{gregorian}}',
  outputFormat:   'YYYY-MM-DD', // format of hidden input value
});
