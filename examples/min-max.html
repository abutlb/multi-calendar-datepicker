Min / Max Dates

Restrict selectable dates to a specific range — تحديد نطاق للتواريخ القابلة للاختيار

Example 1 — Gregorian · Dynamic range (30 days window)

… → …  (past 10 days · next 20 days)
No date selected yet. — لم يتم الاختيار بعد.
ℹ️ Greyed-out days are outside the allowed range and cannot be clicked. Days before … and after … are disabled.

مثال 2 — هجري · نطاق + تعطيل أيام الجمعة والسبت

لم يتم الاختيار بعد. — Not selected.
ℹ️ التقويم هجري ولكن الحد الأدنى والأقصى يُحدد بالميلادي دائماً. أيام الجمعة (5) والسبت (6) معطّلة لأنها إجازة رسمية.

Code

// Compute a dynamic range relative to today
const today   = new Date();
const minDate = offsetDate(today, -10); // 10 days ago
const maxDate = offsetDate(today, +20); // 20 days from now

// Example 1 — Gregorian
MultiCalendarDatepicker.init('#g_display', {
  calendar:        'gregorian',
  locale:          'en',
  minDate:         minDate, // 'YYYY-MM-DD' string or {year,month,day}
  maxDate:         maxDate,
  outputTarget:    '#g_value',
  resultTarget:   '#g_result',
  resultTemplate: 'Selected: {{gregorian}}',
});

// Example 2 — Hijri + disabled weekdays
MultiCalendarDatepicker.init('#h_display', {
  calendar:          'hijri',
  locale:            'ar',
  dir:               'rtl',
  minDate:           minDate,
  maxDate:           maxDate,
  disabledWeekDays: [5, 6], // 0=Sun … 5=Fri, 6=Sat
  outputTarget:      '#h_value',
  resultTarget:     '#h_result',
  resultTemplate:   'الميلادي: {{gregorian}} | الهجري: {{display}}',
});

Key Options — الخيارات المهمة

minDate
Earliest selectable date (Gregorian ISO string or {year,month,day} object)
أقدم تاريخ يمكن اختياره، دائماً بالميلادي
maxDate
Latest selectable date (same format as minDate)
أحدث تاريخ يمكن اختياره
disabledWeekDays
Array of weekday numbers to disable: 0=Sun, 1=Mon … 6=Sat
أرقام الأيام المعطّلة: 0=الأحد … 5=الجمعة، 6=السبت
disabledDates
Array of specific Gregorian ISO strings to block individually
تواريخ محددة يجب إيقافها (بالميلادي)
💡 Important: minDate / maxDate are always in Gregorian regardless of the displayed calendar. The library converts internally. You never need to convert to Hijri or Jalali.
مهم: minDate و maxDate دائماً بالتقويم الميلادي حتى لو التقويم المعروض هجري أو فارسي. المكتبة تتولى التحويل تلقائياً.
